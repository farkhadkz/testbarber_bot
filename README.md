# AI Booking Bot — AI-негізді жазылу/сату ассистенті

Telegram арқылы жұмыс істейтін, **Claude (Anthropic API)** негізіндегі AI-ассистент. Бизнестің қызметтері мен бағаларын біледі, клиентпен сұхбаттасады, жазылуды (booking) автоматты түрде қабылдап, нәтижесін иесінің Telegram-ына жібереді.

Бұл — IT маманның **AI-автоматтандыру қызметін** клиенттерге сату үшін негіз ретінде жасалған дайын өнім (boilerplate). Бір кодты әр клиентке `config/business.json` файлын ғана өзгерту арқылы 30-60 минут ішінде қайта пайдалануға болады.

---

## 1. Мүмкіндіктер

- Клиенттің сұрағына қызметтер тізімі, бағалар, жұмыс уақыты бойынша табиғи тілде жауап беру.
- Сөйлесу тарихын сақтау (SQLite), контекстті есте сақтайды.
- Клиент дайын болғанда (аты, телефоны, қызмет, уақыты белгілі болса) автоматты түрде **жазылым жасайды** және бизнес иесіне Telegram-ға хабарлама жібереді.
- Бір конфиг-файл арқылы кез келген бизнеске (салон, клиника, оқу орталығы, шеберхана) бейімдеу.
- Толықтай self-hosted — деректер тек сенің серверіңде сақталады.

---

## 2. Орнату (Setup)

### Талаптар
- Node.js 18+ (fetch API қажет)
- Telegram bot токені — [@BotFather](https://t.me/BotFather) арқылы алу
- Anthropic API кілті — https://console.anthropic.com

### Қадамдар

```bash
git clone <бұл репо>
cd ai-booking-bot
npm install
cp .env.example .env
```

`.env` файлын ашып, өз токендеріңді қойыңыз:

```
TELEGRAM_BOT_TOKEN=...
ANTHROPIC_API_KEY=...
OWNER_CHAT_ID=...
```

`OWNER_CHAT_ID` — бизнес иесінің Telegram chat ID. Білу үшін [@userinfobot](https://t.me/userinfobot)-ке жазу жеткілікті.

Іске қосу:

```bash
npm start
```

### Боттың командалары

- `/start` — клиентке сәлемдесу хабарламасы
- `/bookings` — (тек `OWNER_CHAT_ID`) соңғы 20 жазылымды көрсетеді

## 3. Бизнес үшін бейімдеу (`config/business.json`)

Әр жаңа клиент үшін осы файлды ғана өзгертесіз:

```json
{
  "businessName": "Бизнес атауы",
  "language": "kk немесе ru",
  "description": "Бизнес туралы қысқаша мәлімет",
  "workingHours": "Жұмыс уақыты",
  "address": "Мекенжай",
  "services": [
    { "name": "Қызмет атауы", "price": 5000, "durationMin": 30 }
  ],
  "extraInstructions": "AI үшін қосымша ережелер: акциялар, тон, ережелер"
}
```

Кодты өзгертудің қажеті жоқ — бот промптты автоматты түрде осы файлдан құрастырады.

## 4. Деплой (өндіріске шығару)

Ең қарапайым нұсқа — арзан VPS (мысалы, $5/ай) немесе Railway/Render платформасы:

1. Жобаны серверге жүктеу.
2. `npm install --production`
3. `pm2 start src/index.js --name booking-bot` (процесті фонда ұстау үшін)

Әр клиентке жеке `.env` және `config/business.json` файлымен, жеке bot токенімен бөлек инстанс жасалады.

## 5. Бұл өнімді қалай сатуға болады (бизнес-модель)

### Ұсыныс құрылымы

| Пакет | Не кіреді | Баға (бір реттік) | Ай сайын |
|---|---|---|---|
| Старт | 1 платформа (Telegram), 5-7 қызмет, негізгі жазылым логикасы | 250 000 – 400 000 тг | 0 |
| Стандарт | + Instagram/WhatsApp интеграциясы, CRM-ге (Google Sheets/amoCRM) автожазу | 500 000 – 900 000 тг | 50 000 – 100 000 тг |
| Бизнес | + аналитика дашборды, көп тілді қолдау, AI промпт-оптимизация | 1 000 000 – 1 800 000 тг | 100 000 – 250 000 тг |

### Сату скрипті (қысқаша)

1. **Аудитория:** Instagram/Kaspi-де белсенді сұлулық салондары, тіс емханалары, оқу орталықтары, шеберханалар — DM-ге жауап беруге уақыты жоқтар.
2. **Демо:** осы бот негізінде нақты клиенттің қызметтерімен 15 минутта демо жасап, Telegram/Instagram-ға жіберу. "Мынау сіздің клиенттеріңізге 24/7 жауап береді және жазылым алады" дегенді көрсету.
3. **Ұсыныс:** "Старт" пакетін 1 апта ішінде орнатуға кепілдік беріп, бірінші 3-5 клиентке жеңілдетілген бағамен (150 000-200 000 тг) ұсыну — портфолио мен пікір алу үшін.
4. **Масштабтау:** дайын кодты template ретінде сақтап, әр жаңа клиентке тек `business.json`-ды өзгерту арқылы жеткізу уақытын 1 күнге дейін қысқарту.

### 30 млн теңгеге жету математикасы

- 10 клиент × ~500 000 тг (орнату) = 5 000 000 тг
- 10 клиент × ~100 000 тг/ай (қолдау) × 2 ай = 2 000 000 тг
- Қалғанын 1-2 ірі "Бизнес" пакетімен (1 000 000+ тг) жабу

Бұл сандар — конверсия 5-10% болатын 100-150 суық хабарламамен апта сайын жетімді көрсеткіш.
